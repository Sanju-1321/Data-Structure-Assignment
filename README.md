# Data Structure Assignment

This repository contains my Data Structures assignment implemented in C.

 Description

The program demonstrates the use of a **stack** to reverse a string.
It uses push and pop operations to store characters and retrieve them in reverse order.

 Concepts Used

* Stack (Array Implementation)
* Push Operation
* Pop Operation
* String Handling in C

 How It Works

1. User enters a string.
2. Each character is pushed into the stack.
3. Characters are popped from the stack.
4. The reversed string is printed as output.

  Example

Input: Hello
Output: olleH
